安全架构 / SECURITY ARCHITECTURE

网页内新增如何做到只有你能操作

  1. 部署时接入 GitHub OAuth，仅允许 Howe 的 GitHub 用户 ID。
  2. 后台表单提交后，由服务端创建或更新 data/research.js。
  3. GitHub Commit 触发网站自动重新部署。

纯静态页面无法安全保存管理员密码，因此本地首版没有伪造一个看似安全、实际可被绕过的登录框。
